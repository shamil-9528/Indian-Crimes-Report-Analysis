## 📌 Project Overview

The **Indian Crimes Report Analysis Dashboard** is a data visualization project designed to analyze crime data across multiple cities in India.  
The dashboard transforms raw crime data into **interactive insights** to understand crime patterns, victim demographics, and case outcomes.

This project demonstrates how **data analytics and visualization tools** can be used to uncover meaningful insights from large datasets.

# ⚙️ Methodology

## 🧹 Data Preparation

- Imported raw crime dataset into **Power BI**
- Cleaned and processed the data
- Handled **missing values and inconsistencies**
- Prepared dataset for analysis and visualization

---

## 📊 Data Modeling

Created **DAX measures** to calculate key metrics:

- **Total Crimes**
- **Weapon Crime Percentage**
- **Solved Case Count**
- **Average Victim Age**
- **Total Female Victims**

---

## 📈 Data Visualization

Designed an **interactive dashboard** using:

- KPI Cards
- Bar Charts
- Line Charts
- Donut Charts
- Slicers for **Year and Gender filtering**

# 📊 Dashboard Features

- Crime distribution by **city**
- Crime trend analysis by **hour and month**
- **Solved vs Unsolved cases** analysis
- **Victim gender distribution**
- Crime categorization by **crime domain**
- Interactive **filters for year and gender**

---

# 🎯 Project Objectives

- Analyze crime distribution across cities
- Identify crime trends over time
- Understand victim demographics
- Present insights through interactive dashboards

---

# 📌 Key Insights

- Major cities such as **Delhi and Mumbai report the highest number of crimes**.
- **Weapon-related crimes represent a significant percentage** of total cases.
- Crime patterns vary based on **time of day and month**.
- **Victim gender analysis highlights demographic trends**.
